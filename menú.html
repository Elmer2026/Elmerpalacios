Desarrollo Web Full Stack

Dominando el ecosistema digital desde el diseño de interfaces hasta la arquitectura del servidor.

Propósito del Curso

Esta materia proporciona las habilidades técnicas necesarias para construir aplicaciones web modernas, escalables y eficientes, utilizando las herramientas estándar de la industria como Angular, Node.js y patrones de arquitectura avanzada.

Módulo 1

Fundamentos del Desarrollo

  • Roles: Front-End, Back-End y Full Stack.
  • Arquitectura cliente-servidor y SDN en capas.
  • Navegadores, hosting y dominios.
  • Herramientas clave:
    • Control de versiones con Git y GitHub.
    • CI/CD con GitHub Actions.
Módulo 2

Interfaces Web Modernas

  • HTML5 semántico y estructura.
  • CSS3 avanzado y Modelo de Caja.
  • Frameworks: Bootstrap y Tailwind CSS.
  • Accesibilidad Web (WCAG).
  • Diseño adaptativo y Responsive Design.
Módulo 3

Arquitectura Back-End

  • Patrones Creacionales:
    • Singleton, Factory Method.
  • Patrones Estructurales:
    • Adapter, Decorator, Composite.
  • Patrones de Comportamiento:
    • Observer, State.
Módulo 4

Desarrollo con Angular

  • Componentes, templates y servicios.
  • Protocolo HTTP y métodos (GET, POST, etc.).
  • Backend-as-a-Service con Firebase.
  • Formularios reactivos y validaciones.
  • Ruteo y navegación entre vistas.
